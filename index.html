💡 Central de Dicas de HTML e CSS

Passe o mouse por cima dos cartões para revelar as instruções e truques!

🖼️ Centralizar Imagens

Para alinhar o mascote ou fotos no meio da tela, use no CSS:

display: block;
margin: 0 auto;

Evita que ela quebre o layout colando nas bordas!

➖ Separar Postagens

Use a tag <hr> para criar linhas horizontais automáticas de divisão.

Ela ajuda a limpar o design e separar o diário de aulas de forma elegante.

📦 Entendendo as Divs

A tag <div> serve como uma caixa vazia para agrupar e organizar blocos de código.

Dica: Sempre lembre de fechá-la com </div> para não quebrar o site!

💻 Abrir o VS Code

Se tiver travado no GitHub, lembre-se de clonar o repositório e abrir a pasta local correta no VS Code.

Só assim o Live Server vai funcionar direitinho!

💾 Não perca o código

O atalho mais importante no laboratório é o Ctrl + S.

Salve o arquivo a cada alteração ou o navegador não mostrará os novos códigos.
