➖ Separar Postagens

Use a tag <hr> para criar linhas horizontais automáticas de divisão.

Ela ajuda a limpar o design e separar o diário de aulas de forma elegante.

📦 Entendendo as Divs

A tag <div> serve como uma caixa vazia para agrupar e organizar blocos de código.

Dica: Sempre lembre de fechá-la com </div> para não quebrar o site!

🎨 Para que serve a tag <style>?

Ela é a estilista do nosso site! Usamos essa tag para mudar toda a parte visual das páginas.

É dentro dela que definimos cores de fundo, fontes, tamanhos e alinhamentos de um jeito organizado.

🧹 Organizando o seu CSS

A tag <style> deve ficar sempre guardada dentro do cabeçalho do site, ou seja, entre as tags <head> e </head>.

Manter as regras visuais separadas dos textos deixa o código muito mais limpo e fácil de consertar.

🖼️ Como colocar Imagens

Para exibir fotos ou ilustrações, usamos a tag de fechamento automático <img>.

Lembre-se de preencher o atributo src="" com o nome exato do arquivo salvo na pasta do seu projeto.

📺 Como colocar Vídeos

A forma mais leve de adicionar um vídeo do YouTube é copiando o código gerado no botão Compartilhar ➡️ Incorporar.

Ele gera uma tag chamada <iframe> que roda o player de mídia perfeitamente na página do blog.

🔄 Carrossel de Vídeos

Para colocar vídeos ou podcasts lado a lado sem esmagar a tela, crie uma caixa <div> organizadora.

Adicione os comandos de estilo display: flex; e overflow-x: auto; para criar a barra de rolagem.
